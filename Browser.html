Line Mode Browser

One of several browsers being developed as part of the CERN WWW project (HyperText) . The LineMode Browser is suitable for use on dumb terminals, requiring no control sequences except for carriage return and line feed. It is also of course useable from terminal emulators in workstation windows. See :
